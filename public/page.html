Explication

Site 'test' ne servant strictement à rien.

Créé pour l'apprentissage techno.web et Node.js

Utilisation des ressources suivantes :
  • Modules 'http', 'fs', 'path' et 'url' de node.js
  • Module nedb.js pour le stockage des informations de connexions
  • Git Hub pour la gestion des versions
  • Open Shift pour l'hébergement

Procédure de déploiement :
  • 'Pusher' l'appli.sur GitHub
  • Sur le site d'OpenShift, 'log in'
  • Puis 'Create your first application now'
  • Puis 'Choose a type of application' : 'Node.js 0.10'
  • Modifier la 'Public URL' : entrer 'sap' à la place de 'nodejs'
  • Dans GitHub, copier l'URL de l'application et la coller dans 'Source Code'
  • Puis 'Create application'
